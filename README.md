# RoamerRadar

RoamerRadar is a travel guide web application for booking hotels, flights, renting cars, and exploring places. Built with Next.js, TypeScript, Tailwind CSS, and Prisma, it provides a modern, full-stack experience with authentication and a rich UI.

## Features

- User authentication (NextAuth.js)
- Browse and book hotels, flights, and car rentals
- List your own property for rent
- Explore destinations with images and reviews
- Responsive, accessible UI with Radix UI and Tailwind CSS
- Dark/light theme support
- Image optimization and fast loading

## Tech Stack

- **Frontend:** Next.js (App Router), React, TypeScript, Tailwind CSS, Radix UI
- **Backend:** Next.js API routes, Prisma ORM
- **Database:** (Configure in `.env`)
- **Authentication:** NextAuth.js
- **Other:** Mapbox GL, Lucide React Icons
